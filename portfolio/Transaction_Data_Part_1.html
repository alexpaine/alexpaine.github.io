Transaction Data - Part I: Cleaning and EDA¶

Transaction level data is invaluable to businesses looking to understand revenue. This series will take a take a close look at the Online Retail dataset available on the UCI Machine Learning Repository.

The dataset contains all transactions between 01/12/2009 and 09/12/2011 occurring for a UK-based non-store online retail company. The company mainly sells unique all-occasion gift-ware and many of its' customers are wholesalers.

In [194]:
# define a cumulative plot
def cplot(df, yr, group, value):
  c = df[df['InvoiceDate'].dt.year == yr].groupby(group)[value].sum()\
      .sort_values(ascending=False).reset_index()
  t = df[df['InvoiceDate'].dt.year == yr][value].sum()
  c['prop'] = c[value] / t
  c[yr] = c['prop'].cumsum()
  return c[yr].plot()
